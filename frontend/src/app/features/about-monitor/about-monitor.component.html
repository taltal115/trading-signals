About · Position monitor

scripts/monitor_open_positions.py reads my_positions with status == open, fetches a daily last price (Yahoo / Stooq), evaluates stops, targets, and hold windows, logs WAIT/SELL-style lines, updates last_alert_* on each document, and optionally posts to Slack when the alert kind changes (deduped). When spot hits your stored target or stop, it can mark the row closed with exit_origin: position_monitor (confirm fills in your broker; signal-only, no orders).

  • Workflow: .github/workflows/position-monitor.yml
  • Secrets: Firestore JSON, optional Slack, optional MONITOR_OWNER_UID.
  • Indicative only — not real-time execution.

Local

PYTHONPATH=./src python scripts/monitor_open_positions.py --config config.yaml
