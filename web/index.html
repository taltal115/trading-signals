About

This project is signal-only: no auto-trading. Summaries below mirror the main docs for the three operational areas.

  • Bot run (scan) — hourly/daily momentum scan, Slack, Firestore signals.
  • Universe discovery — Finnhub → filters → Firestore universe/{asof_date}.
  • Position monitor — GitHub Action, open positions, bracket alerts.

About · Bot run (scan)

The trading bot scans US stocks on daily candles for breakout-momentum setups (1–7 day swing ideas), logs to SQLite, and can post to Slack. It does not place orders.

  • Signal-only: BUY / WAIT / SELL-style guidance with confidence.
  • Manual execution: you decide whether to trade.
  • Outputs: colored terminal logs, ./data/signals.db, optional Slack.

With Firestore enabled, BUY payloads are archived to the signals collection (shown on the Signals page).

Run locally

./run.sh

Ensure config.yaml, universe source (Firestore discovery or static list), and optional .env for Slack / API keys. See repo README.md and docs/bot-logic-and-strategy.md.

About · Universe discovery

Discovery builds a dynamic universe from Finnhub symbols plus your strategy filters, then writes one document per day to Firestore: universe/{asof_date} with symbols, ts_utc, and source.

  • Requires FINNHUB_API_KEY and Firestore credentials.
  • Scheduled via GitHub Actions: .github/workflows/universe-discovery-daily.yml (weekdays), with artifact rotation state.
  • The main scan loads that universe via read_universe_for_date (today or latest fallback).

Local discovery

./run.sh discovery --max-calls 400 --limit 500

About · Position monitor

scripts/monitor_open_positions.py reads my_positions with status == open, fetches a daily last price (Yahoo / Stooq), evaluates stops, targets, and hold windows, logs WAIT/SELL-style lines, updates last_alert_* on each document, and optionally posts to Slack when the alert kind changes (deduped).

  • Workflow: .github/workflows/position-monitor.yml
  • Secrets: Firestore JSON, optional Slack, optional MONITOR_OWNER_UID.
  • Indicative only — not real-time execution.

Local

PYTHONPATH=./src python scripts/monitor_open_positions.py --config config.yaml
